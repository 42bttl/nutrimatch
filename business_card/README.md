# 케어브릿지헬스 명함

**규격**: 90 × 50 mm (한국 표준 명함)

## 파일 구성

| 파일 | 용도 |
|------|------|
| `business_card.html` | 인쇄용 명함 시안 (옵션 A·B 앞뒷면 포함) |
| `logo.svg` | 로고 아이콘 단독 파일 (인쇄소 제출) |

## 인쇄 방법

1. `business_card.html` 을 브라우저(Chrome 권장)에서 열기
2. **Ctrl+P** (인쇄) → 아래 설정 적용:
   - 배경 그래픽: **체크**
   - 여백: **없음**
   - 크기: **실제 크기** (100%)
3. PDF로 저장하거나 바로 인쇄

## 인쇄소 의뢰 시

- `logo.svg` 첨부 (벡터 파일)
- 도무송 재단선: 90 × 50 mm, 모서리 라운드 2 mm 요청
- 용지: 아트지 350g 또는 고급 명함지 추천

## 슬로건 후보

1. **건강을 잇다, 삶을 바꾸다** ← 현재 적용
2. 당신의 건강을 잇는 다리
3. 연결이 만드는 건강한 세상
